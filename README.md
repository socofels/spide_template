# sample
我的基础模板,项目骨架

├─bin  项目文件 
├─dist 存放文档  
├─docs  其他语言的代码  
├─extras  其他脚本库
├─sample.egg-info  测试单元  
└─spide  项目的文件 
    ├─tests  测试文件
    └─units  脚本组件
LICENSE 声明文件  
README  说明文档,自述文件，里面至少得写：项目的特色各种  
requirements.txt  依赖库的列表,依赖的包，用于python方便安装,可以在项目做完后使用`pipreqs，pipreqs . --encoding=utf8 --force`  
setup.py  打包的文件
